------------------------------------------------------------------------------
■動作環境
------------------------------------------------------------------------------
　これをダウンロードできる環境にあるPCなら問題なく動くと思われます。
------------------------------------------------------------------------------
■インストール～起動方法
------------------------------------------------------------------------------
　ダウンロードしたアーカイブを好きな場所に解凍し
　YukkuriMarichanoDaibouken.exeを起動してください。
　フォルダ内部のファイルを削除、移動するとゲームが起動できなくなります。

------------------------------------------------------------------------------
■アンインストール方法
------------------------------------------------------------------------------
　解凍したファイルを削除して下さい。

------------------------------------------------------------------------------
■ストーリー
------------------------------------------------------------------------------
　ゆっくりは、意識を持ち動くおまんじゅう。
　そんなゆっくりである「まりちゃ」は好奇心いっぱい。
　「えいっゆんっ」になることを夢見て、今日も冒険に出かけます…。

------------------------------------------------------------------------------
■操作方法
------------------------------------------------------------------------------
・終了方法
    -Alt+F4キー、もしくはウィンドウのＸボタンで終了できます。


・←→キーまたはADキー：左右への移動

・↑キーまたはｗキー:ジャンプ

・Pキーまたは画面上の一時停止ボタンをクリック：ポーズ
(一時停止時)：一時停止の解除
